# Speak2Draw-Agent-Studio

一个面向“AI 语音绘图工具”议题的项目。用户通过语音指令完成绘图创作，项目重点关注语音指令理解、绘图操作执行、容错处理、响应延迟和复杂指令拆解。

当前项目通过持续 commit、分支和 PR 的方式推进，避免最后一次性提交全部代码。

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开终端中显示的本地地址。首次使用时，浏览器可能要求用户授予麦克风权限；这是浏览器平台限制。授权后，应用内绘图流程应通过语音完成。

## 可用语音示例

- 画一个红色圆形
- 画一个蓝色矩形
- 画一条黑色线
- 写文字你好
- 选择最后一个图形
- 把它改成黄色
- 向右移动一点
- 放大一点
- 撤销
- 重做
- 清空画布
- 导出图片
- 画一个房子和太阳

## 开发规则

- 所有开发必须基于 Git 记录推进。
- 新功能必须通过独立分支和 PR 添加。
- 每个 PR 只做一件事。
- PR 描述必须包含标题、功能描述、实现思路、测试方式。
- main 分支必须始终保持可运行状态。
- 不要提交 `.env`、token、密钥、账号密码等敏感信息。
